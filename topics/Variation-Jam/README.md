# FROGFROGFROG: DANTE'S FROG

Benjamin Merhi

[View this project online] : 

## Description

This project features various variations of the frog game, originally designed by Pippin Barr, and then modded by Benjamin Merhi.

There are a total of 3 new variations:

-Anti-Aircraft Frog

The user controls a frog that shoots rockets and has to shoot down planes to earn points

-Sword Frog

A multiplayer 1v1 game wherein two frogs battle it out with their tongues

-Allegory of The Frog

A puzzle-ish game where you must find enlightenment
